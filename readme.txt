Visual website content builder. Simply drag and drop elements to build your web page content.


== Available Elements ==
* Button
* Empty Space
* Group
* Heading
* Icon
* Image
* Image Compare
* Image Carousel
* Message Box
* Panel
* Progress Bar
* Raw CSS
* Raw HTML
* Raw JS
* Separator
* Stage
* Tabs
* Text Block


= Manual Installation =

1. Download the PageEditor plugin.
2. Upload the `page-editor` folder to the `/wp-content/plugins/` directory of your WordPress.
3. Activate the PageEditor plugin from the 'Plugins' page in WordPress.

= 1.0.0 =
* Initial version of page builder.
* Launched from "Edit with PageEditor" button - available from WP Page/Post edit page.
* Provides a drag and drop environment for elements within the page area.
* Elements can be modified from the properties window by changing the displayed settings.
* Element settings are defined by the applicable module manifesto, which describes the available settings for an element instance and how the module should interface with PageEditor.
* Element Modules: Button, Empty Space, Heading, Icon, Image, Image Compare, Image Carousel, Message Box, Panel, Progress Bar, Raw CSS, Raw HTML, Raw JS, Separator, Stage, Tabs, Text Block.
* New elements can be created by clicking the plus icon, which adds the generated content to the closest element container.
* Feedback and suggestions can be sent to PageEditor developers using the feedback modal - which is displayed by clicking the light-bulb icon in the PageEditor top bar.

== Frequently Asked Questions ==

= How much does it cost? =
Nothing. PageEditor is free for everyone.

= Can I use PageEditor in a project I want to sell? =
Yes. PageEditor is free for commercial use as well as personal use.
If you are a WordPress theme developer, you may include PageEditor bundled with your WordPress theme (which you may sell).
If you are a WordPress hosting provider, you may include PageEditor as part of your service (which you may sell).

= How do I get a new module or feature added to PageEditor? =
We're keen to keep improving PageEditor.
If you have a suggestion that could make PageEditor even better, submit your feedback by clicking the light-bulb icon in the side bar whilst editing a page with PageEditor.
